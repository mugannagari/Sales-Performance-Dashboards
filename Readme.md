# Project Overview

Analyze sales data to identify to key revenue drivers, regional performance and trends to support business decision making. 

# Tools

 - Excel 
 - Pivot table 
 - Visualizations
 - Slicers

# Key Insights 

- West region generates the highest revenue 
- Monitor is the top-performing product 
- Sales show fluctuations across months 
- Seasonal trends are visible in the monthly analysis

# Recommandations

- Promote underperforming products like headphones using discounts. 
- Focus on weaker regions with targeted marketing and regional offers. 
- Running monthly consistence of sales and promotional campaigns.

# Outcome 

Built an interactive dashboards that helps analyse sales performance and supports business decisions .
